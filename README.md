# rn-bitmovin-player

Bitmovin player React Native module

## Installation

```sh
npm install rn-bitmovin-player
```

## Usage

```js
import BitmovinPlayer from "rn-bitmovin-player";

// ...

const result = await BitmovinPlayer.multiply(3, 7);
```
